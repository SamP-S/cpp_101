# C++ 101
## Summary
A set of implementations of fundamental computer science data structures and algorithms implemented in C++11.

## Requirements
- Compile for g++
- No std libs
- No external libs
- Malloc memory management
- Always `int` data
- Testing at scientific orders of magnitude until timeout
